# 地点标记
--------

## 配置文件新增的内容
- 插件命令
- 路径点存储位置
- 传送命令
- 坐标取整

## 一个服务端路点管理器

灵感来源于LucunJi的[Locations](https://github.com/TISUnion/Locations)插件，并且兼容LucunJi的Locations插件数据

标记数据文件路径：`config/location_marker/locations.json`

标记配置文件路径：`config/location_marker/config.json`

## 需求

- [MCDReforged](https://github.com/Fallen-Breath/MCDReforged) >= 2.0.0
- [Minecraft 数据API](https://github.com/MCDReforged/MinecraftDataAPI)
